## 家庭 AI 助手（第一版）

本仓库为“家庭内部使用”的私有 App 第一版（MVP）。

### 目录
- `backend/`：FastAPI + PostgreSQL + 文件存储抽象 + 管理后台（网页）
- `ios_app/`：Flutter iOS 客户端（后续阶段生成）
- `docker/`：部署相关（docker-compose 等）

### 第一版硬约束（务必遵守）
- **不允许用户注册**
- **管理员后台创建账号/禁用账号/重置密码**
- **账号密码登录**
- **ChatGPT / DeepSeek 模型切换（服务端统一封装）**
- **联网搜索开关默认关闭，且只有管理员允许的账号才能开启**
- **附件上传与会话绑定（<=10MB，单次1个，单会话<=5个）**
- **历史会话可进入继续聊天并保留附件记录**
- **API Key 只放后端环境变量**
