## 第一版测试清单（覆盖 10 个硬约束点）

### 账号体系（不能注册）
- [ ] App 无“注册入口/注册按钮/注册页”
- [ ] 后端无 `/register` 类 API（仅 `/api/auth/login`）

### 管理员后台（必须可用）
- [ ] 管理后台可登录：`/admin/login`
- [ ] 管理员可创建普通用户账号（username+临时密码）
- [ ] 管理员可重置密码（用户下次改密：`must_change_password=true`）
- [ ] 管理员可禁用账号（禁用后登录失败/请求被拒）
- [ ] 管理员可删除账号
- [ ] 管理员可配置权限：允许联网搜索、允许上传附件
- [ ] 列表展示：最近登录时间、登录次数、消息次数

### 登录（账号密码）
- [ ] 正确账号密码登录成功
- [ ] 错误密码登录失败（401）
- [ ] 禁用账号登录失败（403）
- [ ] 退出登录后，App 回到登录页

### 模型切换（ChatGPT / DeepSeek）
- [ ] 聊天页可切换模型（下拉）
- [ ] 切换后发送消息，后端 usage_event 记录模型字段

### 联网搜索开关（默认关闭）
- [ ] 默认关闭（首次打开聊天页/新建会话时开关为 off）
- [ ] 未授权账号：开关置灰/不可用
- [ ] 授权账号：开关可打开，但仅 ChatGPT 路线生效
- [ ] DeepSeek 模式下，即使开关打开也应强制为 false（服务端门控）

### 附件上传
- [ ] 允许类型：PDF/DOCX/TXT/JPG/PNG/WEBP
- [ ] 单文件 >10MB：前端提示/后端拒绝（422/413）
- [ ] 单次只允许 1 个附件
- [ ] 单会话最多 5 个附件：第 6 个上传被拒绝
- [ ] 上传后在会话内显示附件列表，可删除

### 附件问答
- [ ] TXT 上传后可提问，回答应基于文本
- [ ] PDF/DOCX 上传后可提问（提取文本后参与上下文）
- [ ] 图片附件：ChatGPT 可问答；DeepSeek 返回明确错误提示切换模型

### 历史记录
- [ ] 自动保存会话与消息
- [ ] 历史页展示会话列表
- [ ] 进入历史会话后可继续聊天
- [ ] 删除会话后不再出现在列表
- [ ] 历史会话中仍能看到附件记录

### API Key 只放后端
- [ ] iOS 工程中不出现 OPENAI/DEEPSEEK key
- [ ] 后端从环境变量读取 key；客户端只调用后端 API
